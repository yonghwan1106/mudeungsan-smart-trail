## 3. UI/UX 상세 개선점

### 랜딩 페이지 (`app/page.tsx`)
- ✅ 카운터 애니메이션 작동 양호
- ⚠️ CTA 버튼 hover 효과 더 강화 필요
- ⚠️ Hero 섹션 배경 이미지/영상 추가 고려

### 대시보드 (`app/dashboard/page.tsx`)
- ✅ 전체적인 레이아웃 양호
- ⚠️ 탐방로 카드 정렬이 불규칙할 수 있음 (grid 고정 높이 필요)
- ⚠️ 트렌드 화살표 색상 반대로 표시될 수 있음 (감소가 긍정적일 때)

### 히트맵 (`app/heatmap/page.tsx`)
- ✅ 실시간 펄스 애니메이션 효과적
- ⚠️ 무등산 실제 지형과 일치하지 않음 (간략화된 SVG)
- ⚠️ 모바일에서 SVG 인터랙션 어려움 (탭 영역 확대 필요)
- 💡 실제 등산로 GPX 데이터 활용 권장

### 예측 페이지 (`app/forecast/page.tsx`)
- ✅ 다양한 차트 타입 제공 (영역, 라인, 막대)
- ⚠️ 피크 시간대 강조 부족
- 💡 예측 vs 실제 비교 기능 추가 고려

### 시뮬레이션 (`app/simulation/page.tsx`)
- ✅ Before/After 비교 직관적
- ⚠️ 슬라이더 인터랙션 추가하면 더 효과적
- 💡 분산 유도 시나리오별 시뮬레이션 옵션

### 알림 센터 (`app/alerts/page.tsx`)
- ✅ 필터링 기능 양호
- ⚠️ 알림음/진동 피드백 없음
- ⚠️ 알림 해결 처리 후 UI 피드백 부족

### 새로 추가된 파일
- `components/Skeleton.tsx` - 로딩 스켈레톤 컴포넌트
- `app/error.tsx` - 에러 페이지
- `app/loading.tsx` - 전역 로딩 상태

### 히트맵 SVG 개선 내용
- 산 지형 그라데이션 배경
- 등고선 (4단계)
- 천왕봉, 서석대, 입석대 봉우리 마커
- 탐방로 입구 마커 (회색 점)
- 나침반 (북쪽 표시)
- 축척바 (1km)
- 선택 시 글로우 효과

### 접근성 개선 내용
- `role="banner"`, `role="navigation"` 등 랜드마크
- `aria-label` 모든 네비게이션 링크
- `aria-current="page"` 현재 페이지 표시
- `focus:ring` 키보드 포커스 스타일
- `aria-hidden="true"` 장식용 아이콘
- `sr-only` 스크린리더 전용 텍스트
